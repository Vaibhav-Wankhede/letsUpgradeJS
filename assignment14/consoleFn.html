Q1. Here we have the Console Functions, also you can check console for Output

console.log("Text") :

This will print the text inside the console.log() function onto the console


console.log(document) :

The entire html document will be shown inside the console


console.log(document.body) :

only body section will be shown inside the console document.Similarly head will show only section


console.error("Some error Occured! ") :

This will show an error on the console


console.warn("This method is already done") :

This will show a warning on the console


console.timeEnd("Time Taken:") :

This will print the exact time required to load the JavaScript content on Console
